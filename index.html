    POLOC

    About

    Poloc geolocates a current user, using the latitude and longitude parameter to search the Instagram API for nearby pictures. Furthermore, pictures are replotted back to the map so the user can pinpoint exactly where those pictures were taken. Poloc allows a user to save those pictures for later viewing and discovery.

    Poloc
        Technologies:
        Javascript, MongoDB, Mongoose, Express, Node, Google Maps API, Geolocation, Instagram API
        Responsibilities:
        This was an individual project, I developed the concept, wireframe, user narrative, coded the front-end and created the database on the back-end.
    Poloc

    Summary

    PECKBOX

    Statement

    PeckBox is an event organizer app. PeckBox enables a user to view all events created & RSVP in a grid-like chronological format. RSVP users will be able to leave status messages. Peckbox allows a user to view each RSVP events with the knowledge of each status of every other user.

    PeckBox
        Technologies:
        Javascript, Angular JS, Mongoose, MongoDB, Express, Google Place API
        Responsibilities:
